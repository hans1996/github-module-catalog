# Taxonomy v2 Implementation Plan

> Execution note: implement the plan task by task and verify each stage before continuing.

**Goal:** Publish a backward-compatible hierarchical Taxonomy v2 with precise leaf capabilities, ancestor inference, and a compact public browse experience.

**Architecture:** Keep `CapabilityAssertion` as the stable public classification contract. Extend rule matching and add ancestor closure in the deterministic classifier, publish a lightweight capability hierarchy in `CatalogManifest`, and use it to generate and validate a detailed taxonomy page while keeping the root README compact.

**Tech Stack:** Python 3.12, Pydantic v2, PyYAML, Typer, pytest, Ruff, mypy, standard-library safe publisher.
